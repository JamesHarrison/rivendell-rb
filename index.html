rivendell-rb by JamesHarrison

Ruby wrappers for the database used by the Rivendell Broadcast Automation System

The Rivendell broadcast automation system is an awesome bit of kit, but as a C++ project it's tricky to hack on and build little scripts around. This set of Ruby wrappers allows for very simple programmatic access to the database that underlies Rivendell, allowing users to build more complex automation systems that fit their needs.

Dependencies

Rivendell (not required, but you need to specify a database by providing /etc/rd.conf) MySQL (development headers) Ruby and RubyGems

Install

gem install rivendell
You're done!
